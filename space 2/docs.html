🎮 Uživatelský manuál

Space Invaders je 2D arkádová střílečka naprogramovaná v Pythonu. Cílem hry je přežít co nejdéle, zlikvidovat co nejvíce nepřátel a meteoritů, a zapsat se na přední místa žebříčku. Hra se ovládá výhradně klávesnicí.

Spuštění hry

Hra se spouští příkazem v terminálu. Ujistěte se, že máte ve stejné složce soubory player.png, enemy.png, meteor.png a base.png.

python space_invaders.py

Průběh hry

Po spuštění se zobrazí hlavní menu s možnostmi Start, Settings a Quit. Kliknutím nebo klávesnicí vyberte Start pro zahájení hry. Vaše loď se nachází ve spodní části obrazovky, základna je hned pod ní. Nepřátelé přilétají v pravidelných vlnách shora a pohybují se doleva a doprava. Meteority padají náhodně a mohou se štěpit.

💡 TIP PRO ZAČÁTEČNÍKY Nejprve se naučte uhýbat střelám nepřátel. Powerupy jsou klíčové — štít vám dá 15 sekund nezranitelnosti, auto-střelba zase masivně zvýší průchodnost vln. Prioritizujte jejich sbírání.

Jak hra skončí

Hra skončí, pokud dojdou všechny životy hráče (výchozí hodnota: 3) nebo pokud základna ztratí všech 5 HP. Po skončení hry zadáte své jméno a skóre se uloží do žebříčku.

⚙️ Herní mechaniky

Nepřátelé

Nepřátelé (kosmické lodě) se spawnují ve vlnách. Každý nepřítel má 1–2 HP. Pohybují se horizontálně a při dosažení okraje obrazovky mění směr. Čas od času vystřelí červenou střelu směrem dolů. Při zásahu nepřítele dostanete 15 bodů.

Meteority

Meteority padají náhodně z horní části obrazovky. Větší meteority mají větší poloměr a jsou nebezpečnější. Při zásahu střelou se meteorit velikosti 2 nebo 3 rozštěpí na dva menší meteority pohybující se do stran. Za každý zasažený meteorit (nebo jeho fragment) dostanete 5 bodů.

Základna

Základna se nachází na spodku obrazovky a má výchozích 5 HP. Poškozuje ji přímý zásah nepřítele, střela nepřítele nebo dopad meteoritu. Základna lze opravit powerupem Klíče (+2 HP, max 10 HP). Pokud HP základny klesne na 0, hra okamžitě skončí.

Hvězdné pozadí

Pozadí tvoří 150 animovaných hvězd pohybujících se dolů. Každá hvězda má náhodnou jasnost, rychlost a pozici. Vytvářejí iluzi letu vesmírem (parallax efekt).

⚡ Powerupy

Powerupy padají náhodně z horní části obrazovky jako padající předměty. Hráč je sebere přímým kontaktem. Jsou celkem 4 typy:

⚡

Auto-střelba

Aktivuje automatické střílení každých 200 ms po dobu 10 sekund. Ideální pro husté vlny nepřátel.

🛡️

Štít

Zapne ochranný štít na 15 sekund. Hráč je nezranitelný vůči střelám nepřátel, meteorům i přímým srážkám.

❤️

Život navíc

Přidá hráči +1 život. Nejcennější powerup v pozdních fázích hry, kdy je každý život vzácný.

🔧

Klíč (oprava)

Opraví základnu o +2 HP (maximum je 10 HP). Nutnost, pokud základna trpí častými zásahy.

💡 STRATEGIE Aktivní powerupy jsou vidět v HUD (horní lišta hry). Čas zbývající u štítu a auto-střelby je zobrazen odpočítáváním. Plánujte sbírání powerupů — nevybíhejte za nimi, pokud tím riskujete zásah nepřítele.

🏗️ Technická dokumentace — Architektura

Projekt se skládá ze dvou hlavních částí: herního enginu napsaného v Pythonu a webového frontendového rozhraní ve vanilla HTML/CSS/JS. Obě části sdílejí společný datový soubor highscores.json.

Python

space_invaders.py — hlavní herní soubor. Obsahuje kompletní herní logiku, objekty, smyčku, vykreslování a ukládání výsledků. Přes 1100 řádků kódu.

JSON

highscores.json — databáze výsledků. Každý záznam obsahuje jméno (name), skóre (score), čas hry v sekundách (time) a datum (date).

HTML/JS

index.html, highscores.html, login.html, register.html, docs.html — webové stránky pro zobrazení žebříčku, správu profilu a dokumentaci. Data z highscores.json se načítají async fetchem.

Python

test_game.py — automatické testy. Testuje škálování rozlišení a ukládání/řazení highscores. Výstup s barevnými ✓/✗ markery. Návratový kód 0 = úspěch, 1 = chyba.

Herní objekty (třídy)

Enemy

Nepřítel se horizontálním pohybem, HP (1–2), střílením. Metody: move(), draw(), shoot().

Meteor

Meteorit s poloměrem, rychlostí a směrem. Při zásahu se štěpí. Kolize se počítá přes math.hypot().

Collectible

Padající powerup s typem (autoshoot, shield, heart, wrench). Metody: move(), draw().

Star

Hvězda na pozadí s jasností, rychlostí. Tvoří parallax efekt vesmíru. 150 instancí generovaných při startu.

💻 Klíčové části kódu

Škálování rozlišení

Hra je referenčně navržena pro 800×600 px. Všechny rozměry objektů se dynamicky přepočítávají při změně rozlišení pomocí koeficientů sx a sy:

REF_W, REF_H = 800, 600

def scale_x(): return WIDTH / REF_W
def scale_y(): return HEIGHT / REF_H

# Příklad: hráč má základní rozměry 50×40 px
player_img = pygame.transform.scale(
    pygame.image.load("player.png").convert_alpha(),
    (int(50 * scale_x()), int(40 * scale_y()))
)

Ukládání highscores

Po skončení hry se skóre zapíše do highscores.json. Záznamy se seřadí sestupně. Soubor se bezpečně načte, aktualizuje a zapíše zpět:

def save_highscore(name, score, game_time):
    path = "highscores.json"
    if os.path.exists(path):
        with open(path, "r", encoding="utf-8") as f:
            data = json.load(f)
    else:
        data = {"highscores": []}

    data["highscores"].append({
        "name":  name,
        "score": score,
        "time":  game_time,
        "date":  time.strftime("%d.%m.%Y %H:%M")
    })

    # Seřadit sestupně podle skóre
    data["highscores"].sort(key=lambda x: x["score"], reverse=True)

    with open(path, "w", encoding="utf-8") as f:
        json.dump(data, f, ensure_ascii=False, indent=2)

🌐 Webové rozhraní

Webová část projektu je napsaná v čistém HTML, CSS a JavaScriptu bez jakýchkoliv externích frameworků (React, Vue, Bootstrap). Všechny stránky sdílí stejný brutalistický design systém.

Tato dokumentace.

Deduplikace žebříčku

Web automaticky deduplikuje záznamy — každý hráč se zobrazí pouze jednou se svým nejlepším skóre:

function dedup(list) {
  const map = {};
  for (const e of list) {
    const name = (e.name || '').toLowerCase();
    if (!map[name] || e.score > map[name].score)
      map[name] = e;
  }
  return Object.values(map).sort((a, b) => b.score - a.score);
}

Responsivní design

Na obrazovkách užších než 900 px se boční navigace transformuje na horizontální lištu nahoře. Využívá CSS media queries a flexbox.

🧪 Testování

Soubor test_game.py obsahuje 8 automatických testů ve 2 skupinách. Testy jsou nezávislé na pygame — testují čistou logiku.

Spuštění testů

python test_game.py

Co se testuje

Test 1a

Základní rozlišení 800×600 dává koeficienty sx=1.0, sy=1.0 (žádné škálování).

Test 1b

Rozlišení 1600×900 — hráčova loď se škáluje ze 50×40 na 100×60 px.

Test 1c

Full HD 1920×1080 — základna se škáluje ze 300×60 na 720×108 px.

Test 1d

Všechna podporovaná rozlišení dávají kladné (nenulové) rozměry.

Test 2a

Funkce save_highscore vrátí True při úspěšném uložení.

Test 2b

Soubor highscores.json je vytvořen se správným jménem hráče.

Test 2c

Více záznamů je seřazeno sestupně podle skóre.

Test 2d

Hráč s nejvyšším skóre je skutečně na prvním místě žebříčku.

ℹ️ BEZPEČNÉ TESTOVÁNÍ Testy používají tempfile.TemporaryDirectory() — dočasnou složku, která se po skončení testů automaticky smaže. Produkční soubor highscores.json nikdy není přepsán.

Webová část

Webové stránky jsou statické HTML soubory. Otevřete index.html přímo v prohlížeči nebo spusťte lokální HTTP server pro správné načítání JSON:

# Python lokální server (doporučeno)
python -m http.server 8000
# Poté otevřete: http://localhost:8000
⚠️ POZOR Soubor highscores.json musí být ve stejné složce jako HTML soubory. Přímé otevření index.html v prohlížeči (file://) může blokovat fetch kvůli CORS politice — použijte lokální server.
